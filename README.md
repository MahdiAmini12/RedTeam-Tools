# RedTeam-Tools

Collection of educational **Red Team** and offensive security tools written in Python.

**Focus areas:** Network reconnaissance, Malware techniques, Tool development for ethical hacking.

---

## 📂 Current Tools

- **[Port Scanner](https://github.com/MahdiAmini12/Port-Scanner)** — Simple multi-port scanner with socket

---

## 🎯 Goals of this Repository

- Build practical offensive security tools
- Learn and demonstrate Red Team techniques
- Improve Python tooling and scripting skills
- Share educational resources for cybersecurity students

---

## 🛠️ Tech Stack
- **Language:** Python 3
- **Libraries:** `socket`, `scapy`, `subprocess`, `argparse`
- **Platforms:** Linux
